# Archivos de texto, rutas y encoding

Tres clases estáticas con responsabilidades distintas: `File` opera sobre archivos, `Directory` sobre carpetas, `Path` construye rutas. `Encoding` controla cómo se interpreta el texto.

## File — leer y escribir

```csharp
using System.Text;

// Leer todo el archivo
string texto = File.ReadAllText("datos.txt");
string texto = File.ReadAllText("datos.txt", Encoding.UTF8);

// Escribir (reemplaza el contenido)
File.WriteAllText("salida.txt", "Hola mundo", Encoding.UTF8);

// Agregar al final
File.AppendAllText("log.txt", "Nueva línea\n", Encoding.UTF8);

// Verificar existencia y borrar
if (File.Exists("viejo.txt")) File.Delete("viejo.txt");
```

## Directory — carpetas

```csharp
// Crear (no falla si ya existe)
Directory.CreateDirectory("datos/salidas/2026");

// Listar archivos
string[] todos   = Directory.GetFiles("mis-archivos");
string[] soloTxt = Directory.GetFiles("mis-archivos", "*.txt");

// Verificar y borrar
if (Directory.Exists("temp")) Directory.Delete("temp", recursive: true);
```

## Path — construir rutas

```csharp
// Siempre usar Combine, nunca concatenar con "/" o "\"
string ruta = Path.Combine("datos", "clientes", "enero.txt");

Path.GetFileName(ruta);                  // "enero.txt"
Path.GetFileNameWithoutExtension(ruta);  // "enero"
Path.GetExtension(ruta);                 // ".txt"
Path.GetDirectoryName(ruta);             // "datos/clientes"
Path.GetFullPath(ruta);                  // ruta absoluta
Path.ChangeExtension(ruta, ".bak");      // cambia extensión
Path.GetTempPath();                      // carpeta temporal del sistema
```

## Encoding — por qué importa

Un archivo es bytes. El encoding es la regla para convertir bytes ↔ caracteres. Si leés un UTF-8 como Latin1 (o viceversa), obtenés caracteres rotos: `mañana` → `maÃ±ana`. Ese fenómeno se llama **mojibake**.

```csharp
// Regla práctica: mismo encoding al escribir y al leer
File.WriteAllText("archivo.txt", contenido, Encoding.UTF8);
string texto = File.ReadAllText("archivo.txt", Encoding.UTF8);

// Para la consola — poner al inicio del programa
Console.OutputEncoding = Encoding.UTF8;
Console.InputEncoding  = Encoding.UTF8;
```

| Encoding           | Cuándo usarlo                                    |
|--------------------|--------------------------------------------------|
| `Encoding.UTF8`    | Por defecto hoy en día; soporta cualquier idioma |
| `Encoding.Latin1`  | Archivos heredados en español/europeo            |
| `Encoding.Unicode` | UTF-16; archivos de Windows internos             |
| `Encoding.ASCII`   | Solo inglés sin acentos                          |

## Ejemplo completo

```csharp
using System.IO;
using System.Text;

string carpeta = "datos";
string entrada = Path.Combine(carpeta, "entrada.txt");
string salida  = Path.Combine(carpeta, "salida.txt");

Directory.CreateDirectory(carpeta);

if (File.Exists(entrada)) {
    string texto = File.ReadAllText(entrada, Encoding.UTF8);
    File.WriteAllText(salida, texto.ToUpper(), Encoding.UTF8);
    Console.WriteLine("Procesado.");
} else {
    Console.Error.WriteLine("No existe el archivo de entrada.");
}
```

## Resumen

| Necesidad         | Método                                     |
|-------------------|--------------------------------------------|
| Leer archivo      | `File.ReadAllText(path, encoding)`         |
| Escribir archivo  | `File.WriteAllText(path, texto, encoding)` |
| Agregar al final  | `File.AppendAllText(path, texto, encoding)`|
| Existe archivo    | `File.Exists(path)`                        |
| Borrar archivo    | `File.Delete(path)`                        |
| Crear carpeta     | `Directory.CreateDirectory(path)`          |
| Listar archivos   | `Directory.GetFiles(path, "*.ext")`        |
| Existe carpeta    | `Directory.Exists(path)`                   |
| Borrar carpeta    | `Directory.Delete(path, recursive: true)`  |
| Unir rutas        | `Path.Combine(parte1, parte2, ...)`        |
