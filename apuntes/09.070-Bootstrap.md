# 🔵 1 — ESENCIAL: Layout con Grid

Bootstrap divide la pantalla en **12 columnas**. Vos elegís cuántas ocupa cada elemento.

## Container (contenedor principal)

Siempre arrancás con un `<div class="container">` (o `container-fluid` para ancho completo).

```html
<div class="container">
  <!-- todo va acá -->
</div>

<div class="container-fluid">
  <!-- ocupa todo el ancho de la pantalla -->
</div>
```

| Clase | Ancho max | Cuándo usarla |
|-------|-----------|---------------|
| `.container` | 1320px | Mayoría de layouts |
| `.container-fluid` | 100% | Cuando querés ocupar todo |

## Row & Col (filas y columnas)

```html
<div class="row">
  <div class="col-4">← ocupa 4/12</div>
  <div class="col-8">← ocupa 8/12 → el maestro/detalle clásico</div>
</div>
```

### Columnas responsive (lo que más vas a usar)

```html
<!-- MAESTRO / DETALLE TÍPICO -->
<div class="row">
  <div class="col-12 col-md-4 col-lg-3">
    ← Panel maestro (lista de contactos)
  </div>
  <div class="col-12 col-md-8 col-lg-9">
    ← Panel detalle (contacto seleccionado)
  </div>
</div>
```

| Clase | Breakpoint | Pantalla |
|-------|-----------|----------|
| `col-12` | siempre | 100% en mobile |
| `col-md-4` | ≥768px | 33% en tablet |
| `col-lg-3` | ≥992px | 25% en desktop |

### Gutters (espaciado entre columnas)

```html
<div class="row g-3">  <!-- gap de 1rem entre columnas -->
  <div class="col-6">...</div>
  <div class="col-6">...</div>
</div>
```

| Clase | gap |
|-------|-----|
| `g-0` | 0 |
| `g-1` | 0.25rem |
| `g-2` | 0.5rem |
| `g-3` | 1rem ✅ (el que más vas a usar) |
| `g-4` | 1.5rem |
| `g-5` | 3rem |

## Cards — la cajita que más vas a usar

Útiles para la lista de contactos (maestro) y para el detalle.

```html

  <div class="card-header">
    <i class="bi bi-person"></i> Contactos
  </div>
  <div class="card-body">
    <p class="card-text">Acá va el contenido</p>
  </div>
  <div class="card-footer text-muted">
    <small>Última actualización: hoy</small>
  </div>
</div>
```

```html

    <h6 class="mb-1">Juan Pérez</h6>
    <small class="text-muted">juan@email.com</small>
  </div>
</div>
```

```css
.contact-card { cursor: pointer; transition: all 0.2s; }

  background-color: #f0f4ff;
  border-color: #0d6efd;
  transform: translateX(4px);
}
```

# 🔵 3 — ESENCIAL: Utilities

## Spacing (margin y padding)

Formato: `{propiedad}{lado}-{tamaño}`

```html
<!-- Margin -->
<div class="m-3">m-3: margin 1rem en todos lados</div>
<div class="mt-3">mt-3: margin-top</div>
<div class="mb-3">mb-3: margin-bottom ← el más usado</div>
<div class="ms-3">ms-3: margin-start (left)</div>
<div class="me-3">me-3: margin-end (right)</div>
<div class="mx-3">mx-3: margin horizontal (left + right)</div>
<div class="my-3">my-3: margin vertical (top + bottom)</div>

<!-- Padding -->
<div class="p-3">p-3: padding 1rem</div>
<div class="pt-3 pb-3 ps-3 pe-3 px-3 py-3">igual que margin</div>
```

| Tamaño | rem | px |
|--------|-----|-----|
| `0` | 0 | 0 |
| `1` | 0.25 | 4px |
| `2` | 0.5 | 8px |
| `3` | 1 | 16px ✅ |
| `4` | 1.5 | 24px |
| `5` | 3 | 48px |

> 💡 **Regla de oro**: `mb-3` entre campos de formulario, `mb-2` entre items de lista, `p-3` dentro de cards.

## Flexbox shortcuts

```html

  <button class="btn btn-primary">

  </button>
</div>
```

### Propiedades Flex

| Clase | CSS |
|-------|-----|
| `d-flex` | `display: flex` |
| `justify-content-start/center/end/between/around/evenly` | `justify-content` |
| `align-items-start/center/end/baseline/stretch` | `align-items` |
| `flex-row/flex-column` | `flex-direction` |
| `flex-wrap/flex-nowrap` | `flex-wrap` |
