# Tipos compuestos y colecciones en C#

| Tipo              | Para qué                      | Clave                    |
| :---------------- | :---------------------------- | :----------------------- |
| `T[]`             | Secuencia de tamaño fijo      | Acceso O(1) por índice   |
| `List<T>`         | Secuencia dinámica            | Agrega y quita elementos |
| `Dictionary<K,V>` | Mapa clave → valor            | Búsqueda O(1) por clave  |
| `HashSet<T>`      | Elementos únicos              | Pertenencia eficiente    |
| Tupla `(T1, T2)`  | Agrupar valores temporalmente | Liviana, sin clase       |
| `record`          | Dato con identidad semántica  | Igualdad por valor       |

## Arrays

Tamaño fijo, índices desde `0`, tipo homogéneo:

```csharp
int[] notas = [7, 3, 9, 5, 8];

int primero  = notas[0];    // 7
int ultimo   = notas[^1];   // 8
var primeros = notas[0..3]; // [7, 3, 9]

foreach (int n in notas) Console.WriteLine(n);

Array.Sort(notas);                      // ordena
Array.Reverse(notas);                   // invierte
Array.IndexOf(notas, 7);                // posición
Array.Exists(notas, n => n > 8);        // true/false
var copia = (int[])notas.Clone();       // copia independiente
```

Arrays multidimensionales:

```csharp
int[,] matriz = { { 1, 2 }, { 3, 4 } };
Console.WriteLine(matriz[1, 0]); // 3

int[][] jagged = new int[3][]; // cada fila puede tener distinto largo
```

## List\<T\>

Dinámica, usa `Count` (no `Length`), expone métodos de instancia:

```csharp
List<string> ciudades = ["Tucumán", "Córdoba", "Rosario"];

ciudades.Add("Mendoza");
ciudades.Insert(0, "Buenos Aires");
ciudades.AddRange(["Salta", "Jujuy"]);

ciudades.Remove("Córdoba");
ciudades.RemoveAt(0);
ciudades.RemoveAll(c => c.Length > 7);

bool existe = ciudades.Contains("Salta");
int  pos    = ciudades.IndexOf("Salta");

ciudades.Sort();
ciudades.Reverse();
```

Con LINQ (se cubre en detalle en su propio apunte):

```csharp
var resultado = ciudades
    .Where(c => c.Length > 5)
    .OrderBy(c => c)
    .ToList();
```

## Dictionary\<K, V\>

Pares clave → valor únicos. Acceso O(1):

```csharp
var telefonos = new Dictionary<string, string>
{
    ["Ana"]    = "381-123-4567",
    ["Carlos"] = "381-987-6543"
};

telefonos.Add("Diego", "381-111-2222");  // lanza si ya existe
telefonos["Ana"] = "381-999-8888";       // agrega o actualiza
telefonos.TryAdd("Ana", "otro");         // no hace nada si ya existe

// Acceso seguro (recomendado)
if (telefonos.TryGetValue("Pedro", out var tel))
    Console.WriteLine(tel);

var fallback = telefonos.GetValueOrDefault("Pedro", "Sin número");

telefonos.Remove("Carlos");

foreach (var (nombre, numero) in telefonos)
    Console.WriteLine($"{nombre}: {numero}");
```

## HashSet\<T\>

Elementos únicos, sin orden garantizado, `Contains` en O(1):

```csharp
var frutas = new HashSet<string> { "manzana", "banana", "naranja" };
frutas.Add("banana"); // ignorado — ya existe

frutas.Contains("banana"); // true

var a = new HashSet<int> { 1, 2, 3, 4 };
var b = new HashSet<int> { 3, 4, 5, 6 };

a.UnionWith(b);       // { 1, 2, 3, 4, 5, 6 }
a.IntersectWith(b);   // { 3, 4 }
a.ExceptWith(b);      // { 1, 2 }
```

## Tuplas

Agrupan valores sin definir una clase. Ideales para retornar múltiples valores:

```csharp
var persona = (Nombre: "Ada", Edad: 20);
Console.WriteLine(persona.Nombre); // Ada

static (double Min, double Max) Rango(int[] datos) =>
    (datos.Min(), datos.Max());

var (min, max) = Rango([3, 7, 1, 9]);

// Descartar campo con _
var (_, soloMax) = Rango([3, 7, 1, 9]);
```

## Records

Tipo especial para datos inmutables con igualdad por valor. El compilador genera constructor, `Equals`, `GetHashCode`, `ToString` y deconstrucción:

```csharp
record Persona(string Nombre, int Edad);

var r1 = new Persona("Ana", 25);
var r2 = new Persona("Ana", 25);

Console.WriteLine(r1 == r2);  // true (igualdad por valor)
Console.WriteLine(r1);        // Persona { Nombre = Ana, Edad = 25 }

// Copiar con cambios (expresión with)
var r3 = r1 with { Edad = 26 };
Console.WriteLine(r1.Edad);   // 25 — no se modificó
Console.WriteLine(r3.Edad);   // 26

// Deconstrucción
var (nombre, edad) = r1;
```

Records con herencia + pattern matching:

```csharp
abstract record Figura;
record Circulo(double Radio)            : Figura;
record Rectangulo(double Ancho, double Alto) : Figura;

double Area(Figura f) => f switch {
    Circulo    { Radio: var r }            => Math.PI * r * r,
    Rectangulo { Ancho: var a, Alto: var h } => a * h,
    _ => throw new ArgumentException()
};
```

## Cuándo usar cada uno

```
¿Tamaño fijo, acceso por índice?         → T[]
¿Tamaño dinámico?                        → List<T>
¿Buscar por clave?                       → Dictionary<K,V>
¿Evitar duplicados / operaciones de conjunto? → HashSet<T>
¿Retornar múltiples valores (temporal)?  → Tupla
¿Dato que viaja entre capas del sistema? → record
¿Proyección temporal en LINQ?            → tipo anónimo
```
