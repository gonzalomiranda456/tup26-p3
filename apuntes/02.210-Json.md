# JSON en C#

JSON (*JavaScript Object Notation*) es el formato estándar para intercambiar datos entre programas. Convierte objetos en texto que cualquier lenguaje puede leer.

## Los seis tipos de valores

```json
"texto"          // string — siempre comillas dobles
42 / 3.14 / -7   // number
true / false     // boolean — minúsculas, sin comillas
null             // null — minúsculas, sin comillas
{ "clave": valor }         // object — pares clave:valor entre llaves
[ valor, valor, valor ]    // array — lista ordenada entre corchetes
```

**Reglas de sintaxis frecuentemente olvidadas:**
- Claves de objetos: siempre entre comillas dobles
- Sin coma después del último elemento
- `true`, `false`, `null` en minúsculas, sin comillas
- No hay comentarios en JSON

## Estructura de objetos y arrays

```json
{
  "id": 1,
  "nombre": "Ada Lovelace",
  "activa": true,
  "telefonos": ["+54 381 111-1111", "+54 381 222-2222"],
  "direccion": {
    "ciudad": "Tucumán",
    "pais": "Argentina"
  }
}
```

Array de objetos (estructura más común en APIs):
```json
[
  { "id": 1, "texto": "Estudiar HTTP",  "completada": false },
  { "id": 2, "texto": "Hacer el TP",   "completada": true  }
]
```

## System.Text.Json en C#

```csharp
using System.Text.Json;
```

### Serializar (objeto → JSON)

```csharp
record Tarea(int Id, string Texto, bool Completada);

var tarea = new Tarea(1, "Estudiar HTTP", false);

// Minificado (por defecto)
string json = JsonSerializer.Serialize(tarea);
// {"Id":1,"Texto":"Estudiar HTTP","Completada":false}

// Con formato legible y camelCase
var opts = new JsonSerializerOptions {
    WriteIndented = true,
    PropertyNamingPolicy = JsonNamingPolicy.CamelCase
};
string json2 = JsonSerializer.Serialize(tarea, opts);
// { "id": 1, "texto": "Estudiar HTTP", "completada": false }
```

### Deserializar (JSON → objeto)

```csharp
string json = """{ "id": 1, "texto": "Estudiar HTTP", "completada": false }""";
Tarea? tarea = JsonSerializer.Deserialize<Tarea>(json);

// Listas
List<Tarea>? tareas = JsonSerializer.Deserialize<List<Tarea>>(json);
```

`Deserialize<T>` devuelve `T?` porque puede ser `null` si el JSON es el literal `"null"`.

### Mapeo de tipos

| JSON | C# |
|---|---|
| `string` | `string` |
| `number` (entero) | `int`, `long` |
| `number` (decimal) | `double`, `decimal` |
| `boolean` | `bool` |
| `null` | `null` (tipo nullable `T?`) |
| `object { }` | clase o record |
| `array [ ]` | `List<T>`, `T[]` |

## Casos frecuentes

**Nombres diferentes** entre C# (`PascalCase`) y JSON (`camelCase`):
```csharp
// Opción 1: política de nombres
var opts = new JsonSerializerOptions { PropertyNamingPolicy = JsonNamingPolicy.CamelCase };

// Opción 2: insensible a mayúsculas al deserializar
var opts = new JsonSerializerOptions { PropertyNameCaseInsensitive = true };

// Opción 3: atributo por propiedad
record Tarea([property: JsonPropertyName("task_id")] int Id, string Texto);
```

**Propiedades opcionales** — si no están en el JSON, quedan con el valor por defecto del tipo:
```csharp
string json = """{ "id": 5, "texto": "Nueva" }""";
var t = JsonSerializer.Deserialize<Tarea>(json); // t.Completada == false
```

**Enums como strings:**
```csharp
var opts = new JsonSerializerOptions { Converters = { new JsonStringEnumConverter() } };
// { "prioridad": "Alta" } en lugar de { "prioridad": 2 }
```

**Propiedades extras** en el JSON que no existen en la clase → se ignoran silenciosamente.

## Ejemplo completo: agenda en JSON

```csharp
using System.Text.Json;

const string archivo = "agenda.json";
var opts = new JsonSerializerOptions {
    PropertyNamingPolicy        = JsonNamingPolicy.CamelCase,
    PropertyNameCaseInsensitive = true,
    WriteIndented               = true
};

var agenda = CargarAgenda(archivo);
agenda.Add(new Contacto(agenda.Count + 1, "Grace", "Hopper", ["+54 381 555-5555"]));
GuardarAgenda(agenda, archivo);

List<Contacto> CargarAgenda(string ruta) {
    if (!File.Exists(ruta)) return [];
    return JsonSerializer.Deserialize<List<Contacto>>(File.ReadAllText(ruta), opts) ?? [];
}
void GuardarAgenda(List<Contacto> c, string ruta) =>
    File.WriteAllText(ruta, JsonSerializer.Serialize(c, opts));

record Contacto(int Id, string Nombre, string Apellido, List<string> Telefonos);
```

Archivo generado:
```json
[
  {
    "id": 1,
    "nombre": "Grace",
    "apellido": "Hopper",
    "telefonos": ["+54 381 555-5555"]
  }
]
```

## Resumen

```
Serializar   → JsonSerializer.Serialize(objeto, opts)
Deserializar → JsonSerializer.Deserialize<Tipo>(json, opts)

Opciones útiles:
  WriteIndented              = true    → legible
  PropertyNamingPolicy       = CamelCase → claves en camelCase
  PropertyNameCaseInsensitive = true   → flexible al deserializar
  Converters = { new JsonStringEnumConverter() } → enums como strings
```
