# Entrada y salida por consola — System.Console

`Console` es una clase estática para leer y escribir en la terminal, controlar colores y cursor, y trabajar con los streams estándar.

## Leer y escribir

```csharp
// Escribir
Console.Write("Sin salto");
Console.WriteLine("Con salto");

// Leer
string? linea = Console.ReadLine();         // espera Enter
ConsoleKeyInfo tecla = Console.ReadKey();   // sin Enter, captura la tecla
```

## Colores y cursor

```csharp
Console.ForegroundColor = ConsoleColor.Cyan;
Console.BackgroundColor = ConsoleColor.DarkBlue;
Console.WriteLine("Texto con color");
Console.ResetColor();

Console.Clear();
Console.CursorVisible = false;
Console.SetCursorPosition(col: 10, top: 5);
var (col, row) = Console.GetCursorPosition();
```

## Streams estándar

| Stream          | Descripción                              |
|-----------------|------------------------------------------|
| `Console.In`    | Entrada estándar (teclado o pipe)        |
| `Console.Out`   | Salida estándar (pantalla o redirección) |
| `Console.Error` | Salida de error (siempre a la terminal)  |

```csharp
// Leer toda la entrada (útil con pipe)
string todo = Console.In.ReadToEnd();

// Escribir errores sin contaminar la salida
Console.Error.WriteLine("Algo salió mal");
```

## Redirección desde shell

```bash
dotnet run < entrada.txt        # stdin desde archivo
dotnet run > salida.txt         # stdout a archivo
dotnet run 2> error.txt         # stderr a archivo
dotnet run < entrada.txt | grep "x"
```

## Encoding

```csharp
// Poner al inicio del programa para soportar acentos y ñ
Console.InputEncoding  = System.Text.Encoding.UTF8;
Console.OutputEncoding = System.Text.Encoding.UTF8;
```

## Resumen

| Necesidad             | Método                                            |
|-----------------------|---------------------------------------------------|
| Leer línea            | `Console.ReadLine()`                              |
| Leer tecla sin Enter  | `Console.ReadKey()`                               |
| Escribir sin salto    | `Console.Write(...)`                              |
| Escribir con salto    | `Console.WriteLine(...)`                          |
| Limpiar pantalla      | `Console.Clear()`                                 |
| Cambiar color         | `Console.ForegroundColor`, `Console.BackgroundColor` |
| Restaurar color       | `Console.ResetColor()`                            |
| Mover cursor          | `Console.SetCursorPosition(col, top)`             |
| Mensajes de error     | `Console.Error.WriteLine(...)`                    |
| Leer todo stdin       | `Console.In.ReadToEnd()`                          |
