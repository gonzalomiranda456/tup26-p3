# Control de flujo y manejo de excepciones en C#

| Familia       | Qué hace                         | Ejemplos                      |
| :------------ | :------------------------------- | :---------------------------- |
| Condicionales | Elige qué ejecutar               | `if`, `switch`                |
| Iterativas    | Repite un bloque                 | `while`, `for`, `foreach`     |
| De salto      | Altera el flujo                  | `break`, `continue`, `return` |
| De error      | Maneja situaciones excepcionales | `try/catch/finally`           |

## `if` / `else`

```csharp
if (temperatura > 35)       Console.WriteLine("Calor extremo");
else if (temperatura > 25)  Console.WriteLine("Calor agradable");
else                        Console.WriteLine("Frío");

// Pattern matching con is
if (valor is int n and > 0) Console.WriteLine($"Entero positivo: {n}");
if (nombre is not null)     Console.WriteLine(nombre.ToUpper());
```

## Operador ternario `? :`

```csharp
string categoria = edad >= 18 ? "adulto" : "menor";

// Para múltiples casos, preferir switch expression
string nivel = nota switch {
    >= 9 => "Sobresaliente",
    >= 7 => "Notable",
    >= 6 => "Aprobado",
    _    => "Desaprobado"
};
```

## `switch` expression (forma moderna)

Produce un valor, no necesita `break`:

```csharp
string nombre = diaSemana switch {
    1       => "Lunes",
    2       => "Martes",
    6 or 7  => "Fin de semana",
    _       => "Otro día"
};

// Con pattern matching de tipos y propiedades
static string Describir(object fig) => fig switch {
    Circulo c                          => $"Círculo r={c.Radio}",
    Rectangulo { Ancho: var a, Alto: var h } => $"Rect {a}×{h}",
    null                               => "nulo",
    _                                  => "desconocido"
};

// Pattern de tupla
static string Clima(int temp, bool llueve) => (temp, llueve) switch {
    (> 30, false) => "Soleado",
    (> 30, true)  => "Bochornoso",
    (_, true)     => "Lluvioso",
    _             => "Fresco"
};

// Pattern de lista
static string Analizar(int[] nums) => nums switch {
    []             => "vacía",
    [var x]        => $"un elemento: {x}",
    [var x, var y] => $"dos: {x} y {y}",
    [var f, .., var u] => $"del {f} al {u}"
};
```

## Bucles

```csharp
// while — condición al inicio, puede no ejecutarse
while (intentos < 3) { ... }

// do while — condición al final, se ejecuta al menos una vez
do { Console.Write("Contraseña: "); } while (input != "1234");

// for — con contador explícito
for (int i = 0; i < 10; i++) Console.Write(i);
for (int i = 9; i >= 0; i--) Console.Write(i); // hacia atrás

// foreach — forma idiomática para colecciones
foreach (var fruta in frutas) Console.WriteLine(fruta);

// con índice
foreach (var (fruta, i) in frutas.Select((f, i) => (f, i)))
    Console.WriteLine($"[{i}] {fruta}");

// async
await foreach (var item in ObtenerAsync()) { ... }
```

## Sentencias de salto

```csharp
break;    // salir del bucle o switch
continue; // pasar a la siguiente iteración
return;   // salir del método

// Ejemplo: salir al encontrar
for (int i = 0; i < nums.Length; i++) {
    if (nums[i] == buscar) { posicion = i; break; }
}

// Ejemplo: saltar elementos
foreach (var v in valores) {
    if (v is null) continue;
    Console.WriteLine(v);
}
```

## Excepciones

```csharp
try {
    int resultado = 10 / divisor;
} catch (DivideByZeroException ex) {
    Console.WriteLine($"División por cero: {ex.Message}");
} catch (Exception ex) {
    Console.WriteLine($"Error inesperado: {ex.Message}");
    throw; // relanzar sin perder el stack
} finally {
    // siempre se ejecuta, pase lo que pase
    Console.WriteLine("Operación finalizada");
}
```

**Lanzar excepciones:**

```csharp
throw new ArgumentNullException(nameof(nombre));
throw new InvalidOperationException("Estado incorrecto");
ArgumentNullException.ThrowIfNull(nombre);  // C# 10+
ArgumentOutOfRangeException.ThrowIfNegative(valor);  // C# 8+
```

**Jerarquía típica:**

```
Exception
├── ArgumentException
│   ├── ArgumentNullException
│   └── ArgumentOutOfRangeException
├── InvalidOperationException
├── NotSupportedException
├── IOException
├── FormatException
└── OverflowException
```

**Crear excepciones propias:**

```csharp
public class NegocioException(string mensaje, string codigo)
    : Exception(mensaje) {
    public string Codigo => codigo;
}

throw new NegocioException("Stock insuficiente", "STOCK_001");
```

**Filtros de excepción:**

```csharp
catch (HttpRequestException ex) when (ex.StatusCode == HttpStatusCode.NotFound) {
    Console.WriteLine("Recurso no encontrado");
}
```

## Resumen

| Necesidad                             | Herramienta                   |
| :------------------------------------ | :---------------------------- |
| Condición simple                      | `if` / `else`                 |
| Múltiples casos que producen un valor | `switch` expression           |
| Repetir sin saber cuántas veces       | `while`                       |
| Ejecutar al menos una vez             | `do while`                    |
| Repetir con índice                    | `for`                         |
| Recorrer colección                    | `foreach`                     |
| Salir del bucle                       | `break`                       |
| Saltar iteración                      | `continue`                    |
| Manejar errores en runtime            | `try/catch/finally`           |
| Señalar estado inválido               | `throw new XxxException(...)` |
