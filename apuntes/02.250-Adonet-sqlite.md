# ADO.NET con SQLite en C#

ADO.NET es la capa de acceso a datos de .NET. Trabaja con clases abstractas (`DbConnection`, `DbCommand`, `DbDataReader`) que cada motor implementa concrétamente.

```
Tu código → DbConnection (abstracto)
               ↓
           SqliteConnection / SqlConnection / NpgsqlConnection ...
```

## Setup SQLite

```bash
dotnet add package Microsoft.Data.Sqlite
```

```csharp
using Microsoft.Data.Sqlite;

const string connStr = "Data Source=agenda.db";
using var conn = new SqliteConnection(connStr);
conn.Open();
```

`using` garantiza que la conexión se cierre aunque haya excepciones.

## SQL básico — DDL y DML

```sql
-- DDL: crear tabla
CREATE TABLE IF NOT EXISTS Contactos (
    Id      INTEGER PRIMARY KEY AUTOINCREMENT,
    Nombre  TEXT NOT NULL,
    Email   TEXT
);

-- DML: insertar, actualizar, eliminar
INSERT INTO Contactos (Nombre, Email) VALUES (@nombre, @email);
UPDATE Contactos SET Nombre = @nombre WHERE Id = @id;
DELETE FROM Contactos WHERE Id = @id;

-- DQL: consultar
SELECT Id, Nombre, Email FROM Contactos WHERE Id = @id;
```

## CRUD completo en C#

**INSERT:**
```csharp
using var cmd = conn.CreateCommand();
cmd.CommandText = "INSERT INTO Contactos (Nombre, Email) VALUES (@nombre, @email)";
cmd.Parameters.AddWithValue("@nombre", contacto.Nombre);
cmd.Parameters.AddWithValue("@email", contacto.Email ?? (object)DBNull.Value);
cmd.ExecuteNonQuery();
```

**SELECT:**
```csharp
cmd.CommandText = "SELECT Id, Nombre, Email FROM Contactos";
using var reader = cmd.ExecuteReader();
var lista = new List<Contacto>();
while (reader.Read()) {
    lista.Add(new Contacto {
        Id     = reader.GetInt32(0),
        Nombre = reader.GetString(1),
        Email  = reader.IsDBNull(2) ? null : reader.GetString(2)
    });
}
```

**Siempre usar parámetros** — nunca concatenar strings. La concatenación permite SQL injection:
```csharp
// ✗ Vulnerable
cmd.CommandText = $"SELECT * FROM Contactos WHERE Nombre = '{nombre}'";

// ✓ Seguro
cmd.CommandText = "SELECT * FROM Contactos WHERE Nombre = @nombre";
cmd.Parameters.AddWithValue("@nombre", nombre);
```

## Transacciones

Una transacción agrupa operaciones: todas se aplican o ninguna.

```csharp
using var tx = conn.BeginTransaction();
try {
    // múltiples comandos con cmd.Transaction = tx
    tx.Commit();
} catch {
    tx.Rollback();
    throw;
}
```

## Clase Agenda completa

```csharp
public class AgendaSqlite {
    private readonly string connStr;
    public AgendaSqlite(string db) {
        connStr = $"Data Source={db}";
        using var conn = new SqliteConnection(connStr);
        conn.Open();
        using var cmd = conn.CreateCommand();
        cmd.CommandText = """
            CREATE TABLE IF NOT EXISTS Contactos (
                Id INTEGER PRIMARY KEY AUTOINCREMENT,
                Nombre TEXT NOT NULL,
                Email TEXT
            )
            """;
        cmd.ExecuteNonQuery();
    }

    public void Agregar(Contacto c) {
        using var conn = new SqliteConnection(connStr);
        conn.Open();
        using var cmd = conn.CreateCommand();
        cmd.CommandText = "INSERT INTO Contactos (Nombre, Email) VALUES (@n, @e)";
        cmd.Parameters.AddWithValue("@n", c.Nombre);
        cmd.Parameters.AddWithValue("@e", c.Email ?? (object)DBNull.Value);
        cmd.ExecuteNonQuery();
    }

    public List<Contacto> ObtenerTodos() {
        using var conn = new SqliteConnection(connStr);
        conn.Open();
        using var cmd = conn.CreateCommand();
        cmd.CommandText = "SELECT Id, Nombre, Email FROM Contactos";
        using var r = cmd.ExecuteReader();
        var lista = new List<Contacto>();
        while (r.Read())
            lista.Add(new Contacto { Id = r.GetInt32(0), Nombre = r.GetString(1),
                                     Email = r.IsDBNull(2) ? null : r.GetString(2) });
        return lista;
    }
}
```

## Resumen

| Clase ADO.NET   | Responsabilidad                                                   |
| --------------- | ----------------------------------------------------------------- |
| `DbConnection`  | Abre/cierra la conexión a la BD                                   |
| `DbCommand`     | Ejecuta SQL (`ExecuteNonQuery`, `ExecuteScalar`, `ExecuteReader`) |
| `DbDataReader`  | Itera fila por fila los resultados de un SELECT                   |
| `DbTransaction` | Agrupa operaciones — commit o rollback                            |

Cambiar de SQLite a PostgreSQL = cambiar `SqliteConnection` por `NpgsqlConnection` y el connection string. El resto del código ADO.NET no cambia.
