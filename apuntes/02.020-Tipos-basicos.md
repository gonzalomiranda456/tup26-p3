# Tipos de datos y operadores en C#

## Tipos primitivos principales

| Tipo | Qué representa | Ejemplo |
|---|---|---|
| `int` | Número entero | `20`, `-5`, `0` |
| `double` | Número con decimales | `3.14`, `1.75` |
| `bool` | Valor lógico | `true`, `false` |
| `char` | Un solo carácter | `'A'`, `'ñ'` |
| `string` | Texto | `"Hola"` |

## Literales: cómo reconocer el tipo

| Si el literal... | Tipo |
|---|---|
| entre comillas dobles | `string` |
| entre comillas simples | `char` |
| es `true` o `false` | `bool` |
| solo dígitos enteros | `int` |
| tiene punto decimal | `double` |
| termina en `m` | `decimal` |
| termina en `f` | `float` |
| termina en `L` | `long` |
| empieza con `0x` | entero hexadecimal |
| empieza con `0b` | entero binario |

```csharp
long   poblacion = 5_000_000_000L;
float  temp      = 25.5f;
decimal saldo    = 1500.25m;
int color        = 0xFF_EC_DE;
int flags        = 0b1111_0000;
int millones     = 1_000_000;   // _ es solo separador visual
```

## Enteros

| Tipo | Tamaño | Rango |
|---|---|---|
| `byte` | 1 byte | 0 a 255 |
| `short` | 2 bytes | -32.768 a 32.767 |
| `int` | 4 bytes | ±2.1 mil millones |
| `long` | 8 bytes | ±9.2×10¹⁸ |

En la práctica cotidiana se usa casi siempre `int`.

```csharp
Console.WriteLine(int.MaxValue);  //  2.147.483.647
Console.WriteLine(int.MinValue);  // -2.147.483.648

Math.Abs(-7);           // 7
Math.Max(3, 5);         // 5
Math.Pow(2, 10);        // 1024 (devuelve double)

int.Parse("42");                      // lanza excepción si falla
int.TryParse("42", out int n);        // versión segura, devuelve bool
```

## Punto flotante y decimal

| Tipo | Tamaño | Precisión | Uso |
|---|---|---|---|
| `float` | 4 bytes | ~7 dígitos | Gráficos, cuando el tamaño importa |
| `double` | 8 bytes | ~15 dígitos | Cálculo general |
| `decimal` | 16 bytes | 28-29 dígitos | Dinero, precisión exacta |

`float` y `double` usan IEEE 754 y no representan exactamente todos los decimales:

```csharp
Console.WriteLine(0.1 + 0.2); // 0.30000000000000004
```

`decimal` usa representación interna decimal y evita ese problema:

```csharp
decimal precio = 19.99m;
decimal iva    = precio * 0.21m; // correcto para dinero
```

Métodos útiles:

```csharp
Math.Abs(-3.75);        //  3.75
Math.Round(3.75, 1);    //  3.8
Math.Ceiling(3.2);      //  4
Math.Floor(3.9);        //  3
Math.Truncate(3.9);     //  3 (descarta decimales, no redondea)
```

## Booleanos

Solo puede ser `true` o `false`. Surge de comparaciones u operaciones lógicas:

```csharp
bool esMayor = 10 > 5;  // true

bool y   = true && false; // false — AND
bool o   = true || false; // true  — OR
bool no  = !true;         // false — NOT
```

`&&` y `||` tienen **cortocircuito**: si el primer operando determina el resultado, el segundo no se evalúa:

```csharp
// Si x es 0, (10 / x) nunca se evalúa:
bool ok = (x != 0) && (10 / x > 1);
```

## Caracteres (`char`)

Un solo carácter Unicode entre comillas simples. Distinto de `string`:

```csharp
char letra = 'A';
string s   = "A"; // tipo distinto
```

Secuencias de escape comunes: `'\n'` (nueva línea), `'\t'` (tabulación), `'\\'` (barra), `'\''` (comilla simple).

## Operadores

### Aritméticos

```csharp
10 + 3   // 13
10 - 3   // 7
10 * 3   // 30
10 / 3   // 3  — división entera (descarta decimales)
10 % 3   // 1  — resto

// Para obtener decimales, al menos un operando debe serlo:
(double)10 / 3  // 3.333...
```

### Incremento y decremento

```csharp
int a = 5;
int b = a++;  // b = 5, a = 6  (post: usa el valor actual, luego incrementa)
int c = ++a;  // c = 7, a = 7  (pre: incrementa primero, luego usa)
```

### Asignación compuesta

```csharp
x += 5;   // x = x + 5
x -= 3;   // x = x - 3
x *= 2;   // x = x * 2
x /= 4;   // x = x / 4
x %= 4;   // x = x % 4
```

### Relacionales

```csharp
a == b   // igual
a != b   // distinto
a > b    // mayor
a < b    // menor
a >= b   // mayor o igual
a <= b   // menor o igual
```

`==` sobre tipos por valor compara contenido. Sobre tipos por referencia compara identidad (salvo que el tipo lo sobreescriba, como `string` o `record`).

### Bit a bit

```csharp
int a = 0b1100;  // 12
int b = 0b1010;  // 10

a & b   // 0b1000 = 8  — AND bit a bit
a | b   // 0b1110 = 14 — OR bit a bit
a ^ b   // 0b0110 = 6  — XOR
~a      // invierte todos los bits
a << 1  // desplaza a la izquierda (× 2)
a >> 1  // desplaza a la derecha (÷ 2)
```

## Tipos por valor y por referencia

Al copiar una variable de tipo **valor**, se copia el dato:

```csharp
int a = 10;
int b = a;
b = 20;
Console.WriteLine(a); // 10 — no cambió
```

Al copiar una variable de tipo **referencia**, se copia la dirección (dos variables apuntan al mismo objeto):

```csharp
var lista1 = new List<int> { 1, 2, 3 };
var lista2 = lista1;
lista2.Add(4);
Console.WriteLine(lista1.Count); // 4
```

`string` es un tipo por referencia pero se comporta como valor gracias a su inmutabilidad.

## Operadores para `null`

```csharp
string? nombre = null;

string texto = nombre ?? "Sin nombre";  // valor por defecto si null
nombre ??= "Valor default";             // asigna solo si era null
int? largo = nombre?.Length;            // accede solo si no es null
```

## Conversión de tipos

```csharp
// Implícita (sin pérdida de información)
int i = 10;
double d = i;  // ok

// Explícita (puede haber pérdida)
double precio = 19.99;
int entero = (int)precio;  // 19 — sin redondeo

// Desde texto
int.Parse("42");                      // lanza excepción si falla
int.TryParse("42", out int n);        // devuelve bool

// Entre referencias (devuelve null si falla, no excepción)
string? texto = dato as string;
```

## Resumen

| Necesidad | Herramienta |
|---|---|
| Enteros | `int`, `long`, `short`, `byte` |
| Decimales de uso general | `double` |
| Decimales para dinero | `decimal` |
| Verdadero/falso | `bool` |
| Un carácter | `char` |
| Ausencia de valor | `null`, `int?`, `string?` |
| Conversión segura desde texto | `TryParse` |
| Conversión segura de referencias | `as` |
