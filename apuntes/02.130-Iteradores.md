## Las interfaces


`yield break` detiene el iterador antes de terminar:

```csharp

    foreach (int n in fuente) {
        if (n > max) yield break; // detener
        yield return n;
    }
}
```

## EvaluaciĂ³n diferida (lazy)

Los mĂ©todos con `yield` **no ejecutan nada al llamarlos**. Solo ejecutan cuando se itera el resultado. Esto permite trabajar con secuencias potencialmente infinitas y encadenar transformaciones sin materializar resultados intermedios:

```csharp

    .Select(n => n * n)       // no ejecuta aĂºn
    .Take(5)                  // no ejecuta aĂºn
    .ToList();                // acĂ¡ sĂ­ ejecuta todo
```

## CuĂ¡ndo usar `yield`

- Secuencias que se calculan sobre la marcha (no estĂ¡n guardadas)
- Secuencias potencialmente infinitas
- Evitar materializar toda la colecciĂ³n en memoria cuando solo se necesita una parte
- Recorrer fuentes externas (archivos, bases de datos) lĂ­nea por lĂ­nea
