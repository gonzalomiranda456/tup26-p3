# Funciones, Delegados y Eventos en C#

Las funciones en C# también son **valores**: se pueden guardar en variables, pasar como argumentos y devolver como resultado.

## Parámetros

```csharp
// Por valor (default): se copia, el original no cambia
static void Duplicar(int n) { n *= 2; }

// ref: alias real — los cambios se ven afuera
static void Incrementar(ref int valor, int cantidad) => valor += cantidad;
int x = 10; Incrementar(ref x, 5); // x = 15

// out: retorno adicional — el método DEBE asignarlo
static bool TryParseFecha(string texto, out DateOnly fecha) {
    if (DateOnly.TryParse(texto, out fecha)) return true;
    fecha = default; return false;
}
if (TryParseFecha("2025-06-15", out var fecha))
    Console.WriteLine(fecha);

// in: referencia de solo lectura (evita copia sin permitir modificación)
static double Traza(in Matriz4x4 m) => m.Datos[0] + m.Datos[5];

// Opcionales y nombrados
static string Formatear(string texto, bool mayusculas = false, int max = 100) { ... }
Formatear("texto", mayusculas: true, max: 50);

// params: número variable de argumentos
static int Sumar(params int[] nums) => nums.Sum();
Sumar(1, 2, 3, 4, 5); // 15
```

## Funciones locales

Se declaran dentro de otro método. Solo son visibles ahí y pueden capturar variables del contexto:

```csharp
static List<int> FiltrarPares(IEnumerable<int> fuente, int minimo) {
    bool EsValido(int n) => n >= minimo && n % 2 == 0; // local
    return fuente.Where(EsValido).ToList();
}
```

## Recursión

Requiere siempre un **caso base** para no recurrir infinitamente:

```csharp
static long Factorial(int n) {
    if (n <= 1) return 1;          // caso base
    return n * Factorial(n - 1);   // llamada recursiva
}
```

## Delegados

Un delegado es una variable que guarda una función. Representa una firma (tipos de parámetros y retorno):

```csharp
delegate int Operacion(int a, int b);

Operacion op = (a, b) => a + b;
Console.WriteLine(op(3, 4)); // 7

op = (a, b) => a * b;
Console.WriteLine(op(3, 4)); // 12
```

Delegado multicast: puede apuntar a varios métodos con `+=`:

```csharp
delegate void Notificacion(string msg);

Notificacion n = msg => Console.WriteLine($"[Email] {msg}");
n += msg => Console.WriteLine($"[SMS] {msg}");

n("Pedido confirmado"); // invoca los dos
n -= ...; // desuscribir
```

## `Func<>`, `Action<>` y `Predicate<>`

No hace falta declarar delegados propios:

| Tipo | Firma | Uso |
|---|---|---|
| `Func<T, TResult>` | Recibe T, retorna TResult | Transformar |
| `Func<T1, T2, TResult>` | Recibe T1 y T2, retorna TResult | Combinar |
| `Action<T>` | Recibe T, no retorna | Efecto lateral |
| `Action` | Sin parámetros, no retorna | Callback sin datos |
| `Predicate<T>` | Recibe T, retorna bool | Filtrar/verificar |

```csharp
Func<int, int, int> sumar   = (a, b) => a + b;
Func<string, int>   largo   = s => s.Length;
Action<string>      imprimir = Console.WriteLine;
Predicate<int>      esPar   = n => n % 2 == 0;
```

## Expresiones lambda

Función anónima compacta:

```csharp
Func<int, int>      cuadrado  = x => x * x;
Func<int, int, int> maximo    = (a, b) => a > b ? a : b;
Action              saludar   = () => Console.WriteLine("Hola");

// Con bloque
Func<int, int, int> max = (a, b) => {
    if (a > b) return a;
    return b;
};
```

**Clausura**: una lambda captura la **variable**, no su valor en ese momento:

```csharp
int base_ = 10;
Func<int, int> sumarBase = n => n + base_;
base_ = 20;
Console.WriteLine(sumarBase(5)); // 25 — ve el nuevo valor

// Bug clásico en loops — usar copia local para evitarlo:
for (int i = 0; i < 3; i++) {
    int copia = i;
    acciones.Add(() => Console.WriteLine(copia)); // captura copia, no i
}
```

## Funciones de orden superior

Funciones que reciben o devuelven otras funciones:

```csharp
// Recibir función como parámetro
static List<T> Filtrar<T>(List<T> lista, Predicate<T> criterio) =>
    lista.Where(x => criterio(x)).ToList();

Filtrar(numeros, n => n % 2 == 0);

// Devolver función como resultado
static Func<int, int> CrearMultiplicador(int factor) => n => n * factor;

var doble = CrearMultiplicador(2);
doble(5); // 10
```

## Eventos

Un **evento** permite a un objeto notificar a otros cuando algo ocurre. Es un delegado con restricciones: solo el dueño puede dispararlo; los suscriptores solo pueden usar `+=` y `-=`.

```csharp
// Patrón estándar
public class PedidoCreadoEventArgs(int id, string cliente) : EventArgs {
    public int    Id      => id;
    public string Cliente => cliente;
}

public class SistemaPedidos {
    public event EventHandler<PedidoCreadoEventArgs>? PedidoCreado;

    protected virtual void OnPedidoCreado(PedidoCreadoEventArgs e) =>
        PedidoCreado?.Invoke(this, e);

    public void CrearPedido(string cliente) {
        // ... lógica ...
        OnPedidoCreado(new PedidoCreadoEventArgs(1, cliente));
    }
}

// Suscribir
var sistema = new SistemaPedidos();
sistema.PedidoCreado += (sender, e) =>
    Console.WriteLine($"Nuevo pedido de {e.Cliente}");

sistema.PedidoCreado -= handler; // desuscribir
```

| | Delegado | Evento |
|---|---|---|
| Quién puede invocarlo | Cualquiera | Solo la clase dueña |
| Quién puede asignar (`=`) | Cualquiera | Solo la clase dueña |
| Suscripción (`+=`/`-=`) | Cualquiera | Cualquiera |

## Resumen

| Necesidad | Herramienta |
|---|---|
| Pasar por referencia real | `ref` |
| Retorno adicional (patrón TryXxx) | `out` |
| Referencia de solo lectura | `in` |
| N argumentos variables | `params` |
| Función auxiliar local | función local |
| Guardar una función que retorna valor | `Func<>` |
| Guardar una función sin retorno | `Action<>` |
| Guardar una función que devuelve bool | `Predicate<T>` |
| Función anónima en el lugar | lambda |
| Notificar cambios a múltiples observadores | evento con `EventHandler<T>` |
