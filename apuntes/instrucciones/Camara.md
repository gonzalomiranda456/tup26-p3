Creá una app web autocontenida, mobile-first, en un solo HTML, que abra la cámara al iniciar, permita capturar una foto, ingresar un prompt, enviarla a la API de imágenes de OpenAI para editarla con el modelo gpt-image-2, mostrar el resultado y permitir descargarlo. Guardá la API key y opciones de modelo/calidad/tamaño/formato en localStorage, con una ventana de configuración que se abra si falta la API key.
