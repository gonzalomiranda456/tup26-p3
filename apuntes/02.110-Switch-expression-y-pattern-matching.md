# Switch Expression y Pattern Matching en C#

El **pattern matching** permite examinar un valor y ramificar según su estructura, tipo o contenido. El **switch expression** es la forma principal: produce un valor directamente.

```csharp
// Antes
string resultado;
if (figura is Circulo c) resultado = $"Círculo r={c.Radio}";
else if (figura is Rectangulo r) resultado = $"Rect {r.Ancho}×{r.Alto}";
else resultado = "desconocida";

// Ahora
string resultado = figura switch {
    Circulo    c => $"Círculo r={c.Radio}",
    Rectangulo r => $"Rect {r.Ancho}×{r.Alto}",
    _            => "desconocida"
};
```

| | `switch` instrucción | `switch` expression |
|---|---|---|
| Produce | Sentencias | Un valor |
| `break` | Obligatorio | No existe |
| `default` | Keyword | `_` (wildcard) |
| Exhaustividad | No verificada | Verificada por compilador |

## Tipos de patrones

### Constante

```csharp
string msg = codigo switch {
    200 => "OK",
    404 => "No encontrado",
    500 => "Error del servidor",
    _   => $"Código: {codigo}"
};

// Con or — múltiples valores, mismo resultado
string resp = comando switch {
    "ayuda" or "help" or "?" => MostrarAyuda(),
    "salir" or "exit"        => "Hasta luego",
    _                        => "Comando desconocido"
};
```

### Tipo (type pattern)

```csharp
string desc = valor switch {
    int    n => $"entero: {n}",
    double d => $"decimal: {d:F2}",
    string s => $"texto de {s.Length} chars",
    null     => "nulo",
    _        => $"tipo: {valor.GetType().Name}"
};
```

### Propiedad (property pattern)

```csharp
decimal descuento = pedido switch {
    { Estado: "cancelado" }               => 0,
    { Total: > 10_000, EsUrgente: false } => pedido.Total * 0.15m,
    { Total: > 5_000 }                    => pedido.Total * 0.10m,
    _                                     => 0
};
```

### Relacional y lógico

```csharp
string nivel = nota switch {
    >= 9           => "Sobresaliente",
    >= 7 and < 9   => "Notable",
    >= 6 and < 7   => "Aprobado",
    < 0 or > 10    => throw new ArgumentOutOfRangeException(),
    _              => "Desaprobado"
};
```

### Tupla

```csharp
string clima = (temp, llueve) switch {
    (> 30, false) => "Soleado",
    (> 30, true)  => "Bochornoso",
    (_, true)     => "Lluvioso",
    _             => "Fresco"
};
```

### Lista (list pattern)

```csharp
string analisis = nums switch {
    []                  => "vacía",
    [var x]             => $"un elemento: {x}",
    [var x, var y]      => $"dos: {x} y {y}",
    [var f, .., var u]  => $"del {f} al {u}"
};
```

### Guardia `when`

```csharp
string desc = figura switch {
    Circulo c when c.Radio == 0        => "Punto degenerado",
    Circulo c when c.Radio > 100       => "Círculo enorme",
    Circulo c                          => $"Círculo r={c.Radio}",
    Rectangulo { Ancho: var a, Alto: var h } when a == h => $"Cuadrado {a}",
    _                                  => "Otra figura"
};
```

### Pattern variable con `is`

```csharp
if (valor is int n and > 0)
    Console.WriteLine($"Positivo: {n}");

if (nombre is { Length: > 0 } s)
    Console.WriteLine($"Nombre: {s.ToUpper()}");
```

## Ejemplo completo — jerarquía de records

```csharp
abstract record Figura(string Color);
record Circulo(string Color, double Radio)            : Figura(Color);
record Rectangulo(string Color, double Ancho, double Alto) : Figura(Color);

static double Area(Figura f) => f switch {
    Circulo    c => Math.PI * c.Radio * c.Radio,
    Rectangulo r => r.Ancho * r.Alto,
    _            => throw new ArgumentException($"No soportada: {f.GetType().Name}")
};
```

## Resumen de patrones

| Patrón | Sintaxis | Cuándo |
|---|---|---|
| Constante | `valor => ...` | Comparar contra un literal |
| Tipo | `Tipo t => ...` | Verificar y extraer tipo |
| Propiedad | `{ Prop: valor }` | Ramificar por propiedades |
| Relacional | `> x`, `< x`, `>= x` | Comparar rangos |
| Lógico | `and`, `or`, `not` | Combinar condiciones |
| Tupla | `(v1, v2) => ...` | Múltiples valores a la vez |
| Lista | `[p1, p2, ..]` | Estructura de colección |
| Guardia | `patrón when condición` | Condición extra |
| Variable | `is Tipo v` | Declarar variable inline |
