# LINQ — Language Integrated Query

LINQ permite consultar y transformar colecciones de forma **declarativa** (qué resultado quiero) en lugar de imperativa (cómo hacerlo paso a paso):

```csharp
// Imperativo — 20 líneas con bucles y lógica dispersa
// ...

// LINQ — 4 líneas que se leen como una descripción
var resultado = pedidos
    .Where(p => p.Estado == "aprobado" && p.Fecha >= DateTime.Today.AddDays(-30))
    .DistinctBy(p => p.ClienteId)
    .OrderByDescending(p => p.Monto)
    .Select(p => p.Monto)
    .ToList();
```

## Evaluación diferida

La mayoría de los métodos LINQ **no ejecutan nada al llamarlos**. Solo construyen un plan. La ejecución ocurre cuando se itera el resultado:

```csharp
var query = productos
    .Where(p => p.Precio > 1000)  // no ejecuta
    .OrderBy(p => p.Nombre);      // no ejecuta

// La ejecución ocurre aquí:
foreach (var p in query) { ... }

// O al materializar:
var lista = query.ToList();   // ejecuta y guarda
```

**Métodos que fuerzan la ejecución** (materializan): `ToList()`, `ToArray()`, `ToDictionary()`, `Count()`, `First()`, `Sum()`, `Any()`, `All()`, `foreach`.

## Métodos más usados

### Filtrado

```csharp
productos.Where(p => p.Precio > 1000 && p.Stock > 0)
productos.OfType<Electronico>()           // filtra y castea
productos.Distinct()                      // sin duplicados
productos.DistinctBy(p => p.Nombre)       // sin duplicados por clave
productos.Take(10)                        // primeros 10
productos.Skip(5)                         // saltar los primeros 5
productos.TakeWhile(p => p.Precio < 5000) // mientras se cumpla
```

### Proyección

```csharp
productos.Select(p => p.Nombre)          // transformar cada elemento
productos.Select(p => new { p.Nombre, p.Precio }) // tipo anónimo
productos.SelectMany(p => p.Etiquetas)   // aplanar colecciones anidadas
```

### Ordenamiento

```csharp
productos.OrderBy(p => p.Precio)
productos.OrderByDescending(p => p.Precio)
productos.OrderBy(p => p.Categoria).ThenBy(p => p.Nombre)
```

### Agrupamiento

```csharp
var grupos = productos.GroupBy(p => p.Categoria);

foreach (var g in grupos) {
    Console.WriteLine($"{g.Key}: {g.Count()} productos");
}

// Agrupar en diccionario
var dict = productos
    .GroupBy(p => p.Categoria)
    .ToDictionary(g => g.Key, g => g.ToList());
```

### Joins

```csharp
var result = clientes.Join(
    pedidos,
    c => c.Id,
    p => p.ClienteId,
    (c, p) => new { c.Nombre, p.Monto }
);

// Left join
var leftJoin = clientes
    .GroupJoin(pedidos, c => c.Id, p => p.ClienteId,
        (c, ps) => new { c.Nombre, Pedidos = ps })
    .SelectMany(x => x.Pedidos.DefaultIfEmpty(),
        (c, p) => new { c.Nombre, Monto = p?.Monto ?? 0 });
```

### Agregación

```csharp
productos.Count()
productos.Count(p => p.Stock > 0)
productos.Sum(p => p.Precio)
productos.Average(p => p.Precio)
productos.Min(p => p.Precio)
productos.Max(p => p.Precio)
productos.Aggregate(0m, (acc, p) => acc + p.Precio)
```

### Búsqueda

```csharp
productos.First()                        // primero (excepción si vacío)
productos.FirstOrDefault()               // primero o null
productos.Single()                       // exactamente uno (excepción si no)
productos.SingleOrDefault()
productos.Any()                          // hay alguno?
productos.Any(p => p.Stock == 0)
productos.All(p => p.Precio > 0)
productos.Contains(elemento)
```

### Conversión

```csharp
.ToList()
.ToArray()
.ToDictionary(p => p.Id)
.ToDictionary(p => p.Id, p => p.Nombre)
.ToHashSet()
.ToLookup(p => p.Categoria)   // como dictionary pero admite múltiples valores por clave
```

## Sintaxis de query (alternativa)

```csharp
// Equivalente al encadenamiento de métodos
var result = from p in productos
             where p.Precio > 1000
             orderby p.Precio descending
             select new { p.Nombre, p.Precio };
```

La sintaxis de métodos es más popular y soporta más operaciones. La de query es más legible para joins y agrupaciones complejas.

## Ejemplo completo

```csharp
record Producto(string Nombre, string Categoria, decimal Precio, int Stock);

List<Producto> productos = [
    new("Monitor",  "Hardware", 450_000,  5),
    new("Teclado",  "Hardware",  85_000, 20),
    new("Mouse",    "Hardware",  35_000,  0),
    new("VS Code",  "Software",       0, 99),
    new("Webcam",   "Hardware",  55_000,  8),
];

var resumen = productos
    .Where(p => p.Stock > 0)
    .GroupBy(p => p.Categoria)
    .Select(g => new {
        Categoria = g.Key,
        Total     = g.Sum(p => p.Precio),
        Promedio  = g.Average(p => p.Precio),
        Cantidad  = g.Count()
    })
    .OrderByDescending(g => g.Total);

foreach (var g in resumen)
    Console.WriteLine($"{g.Categoria}: {g.Cantidad} productos, total ${g.Total:N0}");
```

## Resumen de operaciones

| Operación    | Métodos                                                       |
| :----------- | :------------------------------------------------------------ |
| Filtrar      | `Where`, `OfType`, `Distinct`, `Take`, `Skip`                 |
| Transformar  | `Select`, `SelectMany`                                        |
| Ordenar      | `OrderBy`, `OrderByDescending`, `ThenBy`                      |
| Agrupar      | `GroupBy`, `ToLookup`                                         |
| Combinar     | `Join`, `GroupJoin`, `Concat`, `Zip`                          |
| Agregar      | `Count`, `Sum`, `Average`, `Min`, `Max`, `Aggregate`          |
| Buscar       | `First`, `FirstOrDefault`, `Single`, `Any`, `All`, `Contains` |
| Materializar | `ToList`, `ToArray`, `ToDictionary`, `ToHashSet`              |
