# 1er Parcial - martes *12* de mayo de 2026

El parcial consiste en un examen múltiple choice sobre los temas de teoría vistos en clase.

En la carpeta `apuntes` se encuentran los apuntes de cada clase.
En [preguntas.md](preguntas.md) está la base de preguntas para el parcial; no contiene la respuesta correcta, solo el enunciado y las opciones.

El parcial se realizará con esta base de preguntas.

También pueden encontrar las mismas en https://tup26-p3.netlify.app/, donde se pueden visualizar y registrar cuáles son las respuestas correctas para prepararse para el parcial.

En [ESTADO.md](ESTADO.md) está el estado de cada alumno con respecto a los prácticos realizados; esto no es un requisito para rendir el parcial, pero sí es importante para el desarrollo del curso.
